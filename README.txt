This repo contains multiple mini projects using concepts of RAG. Let me give a brief introduction of each project.

qa_with_gemma: In this project question answering of pdf documents done using open source model Google Gemma.
chat with PDF: In this project a chatbot has been implemented based on the pdf inputs provided.
vision.py: In this project a poc created to read bills generated in pdf file and answer questions based on the input the received from text input.
